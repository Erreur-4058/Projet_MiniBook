# 🌐 MNB — MiniBook

**MNB (MiniBook)** est un réseau social nostalgique qui vous replonge directement dans l'ère de **Windows XP** et de **MSN Messenger**. Alliant design rétro et fonctionnalités modernes, MNB est une expérience interactive unique.

🚀 **Découvrez le site en direct :** [https://erreur-4058.github.io/Projet_MiniBook/public/home/index.html](https://erreur-4058.github.io/Projet_MiniBook/public/home/index.html)

---

## ✨ Fonctionnalités

### 🖥️ Interface Windows XP
- **Barres de titre authentiques** : Boutons "Minimiser", "Maximiser" et "Fermer" au style XP.
- **Sons Système** : Retrouvez les sons cultes d’ouverture de session, d’erreur et de corbeille.
- **Favicon Animé** : Une icône de navigateur qui s'anime dynamiquement.

### 📱 Réseau Social Complet
- **Authentification** : Inscription et connexion sécurisées (stockage local).
- **Le Feed** : Publiez des messages et des images, aimez les publications de vos amis.
- **Profil & Paramètres** : Personnalisez votre avatar et votre pseudonyme.
- **Statistiques** : Suivez votre activité sur la plateforme.

---

## 🛠️ Stack Technique

- **Structure** : [HTML5](https://developer.mozilla.org/en-US/docs/Web/HTML) (Sémantique et accessible)
- **Style** : [Vanilla CSS](https://developer.mozilla.org/en-US/docs/Web/CSS) (Variables CSS, Flexbox, Animations)
- **Logique** : [Vanilla JavaScript](https://developer.mozilla.org/en-US/docs/Web/JavaScript) (ES6+)
- **Stockage** : `LocalStorage` pour une expérience persistante sans serveur.

---

## ⚙️ Installation Locale

Pour lancer le projet sur votre machine :

1. Clonez le dépôt :
   ```bash
   git clone https://github.com/Erreur-4058/Projet_MiniBook.git
   ```
2. Accédez au répertoire :
   ```bash
   cd Projet_MiniBook
   ```
3. Ouvrez `index.html` dans votre navigateur préféré (ou utilisez l'extension *Live Server* sur VS Code).

---

## 🔒 Confidentialité
Nous respectons votre vie privée. Les données sont stockées localement dans votre propre navigateur via le `LocalStorage`. Aucune donnée n'est envoyée à un serveur externe.
