Politique de protection des données personnelles

Dernière mise à jour : 11 juin 2026

1. Introduction

Bienvenue sur MiniBook (MNB). La protection de votre vie privée est une priorité pour nous. Cette politique explique comment nous traitons vos données personnelles dans le cadre de ce projet scolaire d'interface homme-machine (IHM).

2. Données collectées

Nous collectons les informations suivantes lors de votre utilisation du site :

  • Informations de compte : Pseudo, adresse e-mail, mot de passe.
  • Profil : Photo de profil.
  • Contenu : Messages (posts) et images que vous publiez.
  • Interactions : Mentions "J'aime" et relations de suivi (followers/following).

3. Utilisation des données

Vos données sont exclusivement utilisées pour :

  • Permettre la création et la gestion de votre compte.
  • Afficher vos publications sur le fil d'actualité.
  • Gérer vos interactions avec les autres utilisateurs.
  • Améliorer l'expérience utilisateur globale du site.

4. Stockage des données

Important : MiniBook est une application client-side. Toutes vos données sont stockées localement dans votre navigateur via le mécanisme du LocalStorage. Aucune donnée n'est envoyée vers un serveur externe ou stockée dans une base de données distante.

Cela signifie que si vous videz le cache de votre navigateur ou changez d'appareil, vous perdrez l'accès à vos données MiniBook sur cet appareil.

5. Partage des données

Nous ne vendons, ne louons et ne partageons aucune de vos données personnelles avec des tiers. Vos données restent strictement confinées à votre environnement de navigation local.

6. Vos droits (RGPD)

Conformément au Règlement Général sur la Protection des Données (RGPD), vous disposez des droits suivants :

  • Droit d'accès et de modification : Vous pouvez modifier vos informations via la page "Paramètres".
  • Droit à l'effacement : Vous pouvez supprimer votre compte ou vos publications à tout moment. La suppression est immédiate et définitive dans votre stockage local.

7. Contact

Pour toute question concernant cette politique, vous pouvez contacter l'équipe de développement via le dépôt du projet.
